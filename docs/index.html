Frontend De Pendientes

  • Angular
  • Bootstrap
  • JWT

En este proyecto, construí el frontend para interactuar con mi API de pendientes en Laravel mediante Angular. Aplicando los conocimientos adquiridos en un curso de Angular, implementé un interceptor para enviar el token obtenido durante el inicio de sesión al backend. Además, establecí un guardian para proteger las rutas y, por defecto, convertí la aplicación en una PWA (Progressive Web App).

Subastas En Linea

  • PHP
  • JavaScript
  • Bootstrap

Este proyecto se centra en la exhibición de subastas directamente en la página principal, brindando a los usuarios la posibilidad de explorar en detalle cada oferta. Al iniciar sesión, los usuarios pueden participar activamente registrando, editando y eliminando subastas. La tecnología utilizada incluye Php, AdminLTE, HTML, CSS, JavaScript, JQuery, MySQL y GitLab, asegurando una plataforma robusta y completa para la gestión y participación en subastas en línea.

Tu Casa Ciclista

  • PHP
  • JavaScript
  • Bootstrap

Junto a mis compañeros de clase, desarrollé el proyecto "Tu Casa Ciclista", una tienda en línea especializada en la venta de bicicletas. La plataforma exhibe detalladamente cada bicicleta y, al iniciar sesión, los usuarios pueden explorar y gestionar productos, permitiéndoles crear, editar y eliminar artículos según sus preferencias. Este proyecto busca ofrecer una experiencia de compra completa y personalizada para los entusiastas de las bicicletas.
